# 빠른 예약 (Race) 사용 가이드

예약 오픈 시각 정각에 자동으로 요청을 발사하는 모드입니다.
주로 매월 25일 22:00(시민 일반 예약 오픈)에 사용합니다.

---

## 5단계 절차

### 1단계 — 계정 등록

1. 하단 탭 **계정 설정** 으로 이동.
2. gytennis 아이디·비밀번호를 입력 후 **로그인**.
3. 세션 활성 상태("✓ 로그인됨")를 확인하고 **빠른 예약** 탭으로 돌아옵니다.

> 자격증명은 이 기기의 브라우저 저장소에만 보관되며 서버에 전송되지 않습니다.

---

### 2단계 — 코트·날짜 선택 및 슬롯 조회

1. **코트장** 드롭다운에서 원하는 코트 단지를 선택합니다.
2. **예약일** 에 예약하려는 날짜를 입력합니다 (기본값: 다음달 같은 날).
3. **슬롯 조회** 버튼을 누릅니다.
4. 슬롯 그리드가 나타나면 예약하려는 슬롯(시간×코트)을 클릭해 선택합니다.
   - ○ = 예약 가능
   - × = 예약 불가

> 일일 한도(보통 2슬롯)·코트당 한도가 그리드 하단에 표시됩니다.

---

### 3단계 — 발사 시각 설정

1. **발사 시각** 입력란에 예약 오픈 시각을 입력합니다.
   - 기본값: 오늘(또는 내일) 22:00
2. **발사 대기** 버튼을 누릅니다.
   - 앱이 자동으로 서버 시계와 동기화한 뒤 카운트다운을 시작합니다.

---

### 4단계 — 카운트다운 대기

- 화면에 **MM:SS.ms** 카운트다운이 표시됩니다.
- **앱을 닫거나 화면을 잠금하지 마세요** (폰 화면 자동 꺼짐 방지 권장).
- 탭을 벗어났다 돌아와도 카운트다운이 자동 재동기됩니다.
- 취소하려면 **취소** 버튼을 누릅니다.

---

### 5단계 — 결제 완료

1. 발사 후 "✅ 예약 성공" 화면이 표시되면 **결제창 열기** 버튼을 누릅니다.
2. KCP 결제창에서 8분 이내에 결제를 완료합니다.
   - 남은 시간이 카운트다운(MM:SS)으로 표시됩니다.
   - 1분 미만이면 빨간색으로 표시됩니다.
3. 결제를 포기하려면 **지금 취소** 버튼을 누릅니다 (슬롯 즉시 해제).

---

## 주의 사항

- 결제창을 닫아도 슬롯이 자동으로 취소되지 않습니다 (R-E B안).
  8분 카운트다운이 만료될 때만 자동 취소됩니다.
- 폰 시계가 크게 어긋난 경우 오차가 발생할 수 있습니다.
  앱이 자동으로 서버 오프셋을 측정하므로 대부분의 경우 보정됩니다.
- 발사 후 "❌ 실패" 화면이 표시되면 **슬롯 재조회** 버튼으로 현재 상황을 다시 확인할 수 있습니다.
