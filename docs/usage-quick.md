# 간편 예약 (Quick) 사용 가이드

예약 가능한 슬롯을 한눈에 보고 탭 한 번으로 예약하는 모드입니다.
이미 오픈된 슬롯이 남아있을 때 빠르게 선점하는 데 적합합니다.

---

## 사용 절차

### 1. 계정 등록

1. 하단 탭 **계정 설정** 으로 이동.
2. gytennis 아이디·비밀번호를 입력 후 **로그인**.
3. **간편 예약** 탭으로 돌아옵니다.

---

### 2. 날짜 선택 및 슬롯 조회

1. **조회 날짜** 입력란에서 예약하려는 날짜를 선택합니다.
2. 우측 **↻** 버튼을 눌러 슬롯을 새로고침합니다.
   - 즐겨찾기에 등록된 코트만 표시됩니다 (미등록 시 전체 코트).

---

### 3. 즐겨찾기 설정 (선택)

- 코트 카드 우측의 **☆ 즐겨찾기** 버튼을 눌러 자주 가는 코트를 등록합니다.
- 등록 후에는 해당 코트만 표시되어 화면이 단순해집니다.

---

### 4. 슬롯 예약

1. 원하는 시간대의 슬롯 버튼을 탭합니다.
2. 확인 대화상자가 나타나면 **확인** 을 눌러 예약을 진행합니다.
3. KCP 결제창이 자동으로 열립니다 — 결제를 완료하세요.

---

### 슬롯 상태 표시

| 표시 | 의미 |
|------|------|
| `{코트번호}번 {시작시간}시` | 예약 가능 — 탭하면 예약 |
| `⏳ {코트}번 {시간}시 결제중` | 방금 예약했으나 결제 진행 중 (노란색) |

---

## 주의 사항

- 결제창은 사용자가 슬롯을 탭한 직후에만 열립니다 (자동 팝업 차단 방지).
- 자격증명은 이 기기의 브라우저 저장소에만 보관되며 서버에 전송되지 않습니다.
- 세션이 만료된 경우 자동으로 재로그인을 시도합니다.
- 결제창을 열기 위해 팝업 차단을 허용해야 할 수 있습니다.
