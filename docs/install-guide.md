# Booking Tennis PWA 설치 안내

고양시 공공테니스장 빠른 예약 도우미. 폰 한 대 + 브라우저 한 개로 끝.

## 설치 URL

배포된 URL을 카톡으로 전달받은 그대로 사파리(아이폰) 또는 크롬(안드로이드)에서 엽니다.

예: `https://booking-tennis-xxxxx.vercel.app`

---

## 아이폰 (Safari)

1. 위 URL을 **Safari** 로 엽니다. (Chrome 으로 열면 동작은 같지만 홈 추가 메뉴가 다릅니다)
2. 하단의 **공유 버튼(⬆)** 탭
3. 메뉴에서 **"홈 화면에 추가"** 선택
4. 이름을 `Tennis` 등으로 짧게 바꾸고 **추가** 탭
5. 홈 화면에 🎾 아이콘이 생깁니다 — 일반 앱처럼 실행

### 주의 (iOS 한정)
- 앱을 닫으면 카운트다운이 멈춥니다. **22:00 정각 발사 시점에는 앱이 열려 있어야** 합니다.
- 백그라운드 알림 부분지원 (iOS 16.4+). 너무 의존하지 마세요.
- 화면 자동 꺼짐 방지: 설정 → 디스플레이 → 자동 잠금 → **사용 안 함** (정각 발사 직전 5분간만)

---

## 안드로이드 (Chrome)

1. URL을 **Chrome** 으로 엽니다.
2. 페이지 하단에 **"홈 화면에 추가"** 배너가 자동으로 뜹니다. 탭.
3. 또는 우측 상단 ⋮ → **"앱 설치"** / **"홈 화면에 추가"** 직접 선택.
4. 이름 확인 후 **추가** 탭.
5. 홈 화면에 🎾 아이콘이 생깁니다.

### 안드로이드 보너스
- 푸시 알림 사용 가능 (추후 기능)
- Wake Lock 지원: 카운트다운 화면이 자동으로 꺼지지 않습니다

---

## 첫 실행

1. 앱 아이콘 탭 → 홈 화면 열림
2. **계정 설정** 카드 탭
3. gytennis 아이디/비번 입력
4. **"이 기기에 저장"** 체크 (다음 접속 시 자동 입력) — 폰 안에만 저장됩니다
5. **로그인** 버튼 → "✅ 로그인됨" 표시되면 성공
6. 홈으로 돌아가서 빠른예약 / 간편예약 사용 시작

---

## 두 가지 사용 모드

### 🚀 빠른 예약 — 매월 25일 22:00 같은 오픈런용

1. 22:00 보다 1~5분 전에 앱 실행
2. **빠른 예약 > 시작**
3. 코트장·예약일·발사시각 설정 (기본값: 다음 22:00, 다음 달 같은 날짜)
4. **슬롯 조회** → 원하는 코트면·시간 슬롯 탭
5. **발사 대기** 버튼 → 카운트다운 시작
6. 정각 도래 → 자동 발사 → 결제창 자동 열림
7. 카드 정보 입력 → 결제 완료

### ⚡ 간편 예약 — 평상시 빈자리 잡기

1. 앱 실행 → **간편 예약 > 열기**
2. 즐겨찾기한 코트들의 빈 슬롯이 한눈에 보임
3. 빈 슬롯 탭 → 결제창 즉시 열림
4. 결제 완료

코트 옆 **☆/★** 토글로 즐겨찾기 추가/삭제.

---

## 보안

- 아이디/비번은 **본인 폰의 브라우저 저장소** 에만 보관됩니다.
- 서버에는 어떤 자격증명도 전송되지 않습니다 (Vercel Function 은 무상태 프록시).
- 본인 외에 다른 사용자가 같은 URL을 써도 **각자의 폰**에 저장된 자기 계정만 쓰입니다.
- "계정 삭제" 버튼으로 즉시 폐기 가능.

---

## 문제 해결

| 증상 | 해결 |
|------|------|
| 결제창 안 열림 | 팝업 차단 해제. 사파리 설정 → 팝업 차단 OFF |
| 슬롯이 안 보임 | 세션 만료 → 계정 설정에서 로그아웃 후 재로그인 |
| 발사 카운트다운 멈춤 | 앱이 백그라운드로 갔음. 정각 시점엔 화면 켜둘 것 |
| "예약 가능"인데 실패 | 다른 사용자가 선점 / 한도 초과 (일 2슬롯, 코트당 1슬롯) |
| 아이콘이 없음 | 홈 화면 추가 안 됨. iOS 는 Safari 로, Android 는 Chrome 으로 |

---

## 결제 PG (KCP) 동작

- 결제는 **KCP (㈜한국사이버결제)** PG 사가 처리합니다.
- 결제창은 별도 탭에서 열립니다.
- 결제 완료 후 자동으로 gytennis 사이트가 갱신됩니다.
- 결제 안 하면 8분 후 자동 취소 → 슬롯 해제.
